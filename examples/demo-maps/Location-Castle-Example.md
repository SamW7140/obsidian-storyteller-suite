# Castle Example

This note demonstrates a simple location marker using frontmatter.

## Features
- Uses `location` frontmatter field with [lat, lng] coordinates
- Will appear as a marker on maps that scan this folder
- Click the marker to open this note

## Usage
Enable "Frontmatter Markers" in plugin settings and specify this folder as a marker source.
