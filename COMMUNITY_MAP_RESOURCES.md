## ğŸ“¥ How to Add Maps to Templates

### **Method 1: Manual Addition**

1. **Download map image** from any source above
2. **Add to Obsidian vault**: Drag & drop into vault folder
3. **Create new map**:
   - `Ctrl+P` â†’ "Create new map"
   - Choose template
   - Name your map
4. **Add background**:
   - Go to "Background" tab
   - Click "Select Image"
   - Choose your downloaded map
5. **Save**

### **Method 2: Set as Template Default** (Advanced)

You can modify `src/utils/MapTemplates.ts` to include paths to community maps:

```typescript
// Example: Add downloaded map to World template
{
    id: 'world-map',
    name: 'World Map',
    // ... other properties
    backgroundImagePath: 'Maps/Templates/azgaar-world-01.png',
    // Or use data URI for embedded images
    backgroundImageData: 'data:image/png;base64,...'
}
```
